Attendance Management System

Mark attendance automatically using your location — fast, secure, and smart.

Why This System?

This application solves the problem of manual attendance by automating the process using geolocation. It ensures only users within a valid area can mark attendance, promoting fairness and accuracy.

Key Features

Geolocation-Based Check-In

Attendance is marked only when a user is within the authorized location circle.

Firebase Integration

Secure real-time database to store attendance data using Google Firebase.

Instant Notifications

Notifies the user if they are outside the valid location during check-in.

Benefits

Accurate

No proxy attendance or manual errors — ensures authenticity.

Efficient

Saves time for both teachers and students by automating the process.

Simple UI

User-friendly interface that works seamlessly across devices.
